# Comunicación Técnica

La comunicación técnica es el puente entre la interpretación subjetiva y el protocolo objetivo.


## Procesos de Comunicación

- **Handshake**  
  Proceso de establecimiento de conexión o acuerdo entre partes.

- **Input de Usuario**  
  Suele ser subjetivo; el sistema debe mitigarlo para evitar ambigüedades.

- **Flujo de Datos**  
  Los servidores solicitan datos a las terminales mediante los **[Endpoint](./Endpoint.md)**, utilizando los **[Socket](./Socket.md)** como canal de transporte.


## Herramientas de Formalización

- **Documentación**  
  Herramienta principal para registrar y estandarizar procesos.

- **Analogías**  
  Transforman la subjetividad en objetividad mediante comparaciones estructuradas.

# Time_Stamping

El **Time Stamping** (Sellado de Tiempo) es un proceso de certificación digital que actúa como un tercero de confianza.

## Definición Técnica

- **Intermediario Jerárquico**  
  Es una entidad o servidor externo que interviene en el envío de datos.

- **Función**  
  Asigna una fecha y hora exacta a un paquete de información, aplicando una firma digital.

## Mecanismo de Seguridad

- **Validación de Integridad**  
  Permite establecer verificaciones matemáticas para asegurar que los datos no fueron modificados tras el sellado.

- **No Repudio**  
  Es una herramienta clave para el **[No_Repudio](./No_Repudio.md)**, ya que el emisor no puede negar la existencia del mensaje en el tiempo registrado.

- **Infracción**  
  Si el sistema detecta un acceso o edición posterior al "sello", la transacción o el documento quedan automáticamente invalidados.

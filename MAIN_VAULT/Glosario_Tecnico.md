# Glosario_Tecnico

- **Angular**  
  Framework de TypeScript para el frontend encargado de gestionar información.

- **Framework**  
  Conjunto de herramientas de conexión para el desarrollo.

- **Punteros**  
  Espacios encargados del almacenamiento o direccionamiento de memoria.

- **Header**  
  Parte inicial de un archivo que contiene información de control.

# Voltaje
Unidad utilizada para expresar la diferencia de potencial eléctrico entre dos puntos.

Se mide en volts (V).

Si ambos puntos tienen el mismo potencial eléctrico:
\[V = 0\]

---
# Ejemplo
Si medimos entre:
- tierra ≈ 0V
- fase ≈ 220V
obtendremos aproximadamente:
\[220V\]
\[220V - 0V = 220V\]

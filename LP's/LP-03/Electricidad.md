# Electricidad
Es un fenomeno fisico que se manifiesta con eventos relacionados a diferentes fuerzas.
generalmente esta se manifiesta mediante el movimiento de cargas electricas (electrones):

+ #### Electricidad estatica
+ #### Electricidad dinamica

## ¿Para que sirve? 
Este concepto es la base de toda tecnologia de hoyen mia (Energgia, Electrcidad, Redes)

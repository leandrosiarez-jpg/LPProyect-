# ¿Como instalar GNS3?

## Requisitos para instalarlo
los requisitos minimos que se vana anecesitar para poder instalar GNS3 son:

-Procesador de 64 bits
-4 GB de RAM (Se recomienda 8 GB o mas)
-Conexion a internet
-VirtualBox o VMware (Opcional)
-1 GB de espacio en el disco
-Habilitar la virtualizacion (VT-x/AMD-V) en la BIOS

## Sistemas Operativos compatibles

-Windows
-Linux
-MacOS

## Pasos de Instlacion 
los pasos para poder tener GNS3 en nuestro dispositivos son:

1. Ingresar a la pagina oficial de GNS3 
2. Descargar a version correspondiente a tu sistema operativo
3. Ejecutar el instalador
4. Aceptar permisos y recusrsos correspondientes
5. Finalizar la instalacion
6. Abrir GNS3 y realizar la instalacion inicial


## Cofiguracion Inicial
para hacer la configuracion inicial al entrar a GNS3 tnemos que:

-Seleccionar servidor local
-Configurar VirtualBox o VMware si se van a usar maquinas virtuales
-Importar imagenes IOS para routers o switches

# Investigación: Puertos y Protocolo TCP

## 1. ¿Qué es un puerto de red?
Es una interfaz lógica o puerto virtual que permite a un equipo organizar y separar diferentes tipos de tráfico de red. Existen 65.535 puertos, numerados del 0 al 65535. Los servicios estándar utilizan números específicos para comunicarse; por ejemplo, el puerto 80 se utiliza para tráfico web (HTTP) y el 22 para conexiones seguras (SSH).

---

## 2. El Protocolo TCP
El *Transmission Control Protocol* (TCP) es uno de los protocolos fundamentales de la capa de transporte en las redes de datos. A diferencia de otros protocolos como UDP, TCP está orientado a la conexión y garantiza que los datos se entreguen completos, en orden y sin errores entre el origen y el destino.

---

## 3. El proceso de conexión
Antes de transferir cualquier dato, TCP realiza un saludo o "Three-Way Handshake" para verificar que el puerto está disponible y la máquina responde. Este proceso consta de tres pasos:

* **SYN:** El cliente envía un segmento inicial de sincronización al puerto del servidor.
* **SYN-ACK:** El servidor responde confirmando la recepción del mensaje y enviando su propia sincronización.
* **ACK:** El cliente envía una última confirmación. En este punto, la conexión queda establecida.

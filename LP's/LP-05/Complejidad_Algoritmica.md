# Complejidad algoritmica 

### ¿que es un algoritmo?

Un algoritmo es una secuencia o serie de pasos precisos para solucionar un problema o llegar a un fin, No necesaria un codigo, cualquier. Este debe contar con ciertas caracteristicas para ser considerado ser un algoritmo.
1. Debe tener un __inicio__ y un __final__.
2. Debe ser conciso, claro y sin ambiguedades.
3. Todo algoritmo debe responder de la misma manera ante el mismo problema.
   
### ¿Cual es el fin de la complejidad algoritmica?

La complejidad algoritmica es una forma de medir la cantidad de Tiempo o Memoria que tarda un algoritmo en ejecutarse inicialmente. Se utiliza para medir la eficiencia del algoritmo.

## ¿Qué es una dirección IP?

Una dirección IP (Internet Protocol) es una etiqueta numérica que identifica de manera lógica y [**jerárquica**](../../../MAIN_VAULT/Seguridad%20y%20Jerarquia.md) a un [**Endpoint**](../../../MAIN_VAULT/Endpoint.md) dentro de una red que utilice el protocolo IP. Siguiendo el modelo OSI, las IPs operan en la Capa 3 (Red).

## ¿Diferentes tipos de IP?

IPv4 e IPv6 son dos versiones del Protocolo de Internet (IP) que permiten la comunicación entre dispositivos a través de redes. Aunque ambos protocolos cumplen la misma función básica, presentan diferencias significativas en su estructura y capacidades.
 
- ###  IPv4 

(Internet Protocol version 4): Creado en los 80, utiliza direcciones de 32 bits. Esto limita el número de direcciones a unos $4.3$ mil millones. Se agotan rápido, por lo que usamos técnicas como NAT (Network Address Translation) para que muchos dispositivos compartan una sola IP pública.

IPv4: 32 bits (4 octetos). Ejemplo: 192.168.1.1.

- ### IPv6

IPv6 (Internet Protocol version 6): Creado para solucionar el agotamiento de IPv4. Usa 128 bits, lo que permite una cantidad de direcciones prácticamente infinita ($3.4 * 10^{38}$).

IPv6: 128 bits (8 grupos hexadecimales). Ejemplo: 2001:0db8:85a3:0000:0000:8a2e:0370:7334.

# Margen Tecnico y Documentacion Oficial

Las explicaciones tecnicas desarrolladas en este proyecto abarcan desde la especificacion formal de lenguajes de modelado hasta herramientas modernas de visualizacion y tecnologias emergentes.

## 1. Modelado de Sistemas (UML)

El Lenguaje de Modelado Unificado (UML) es definido como un lenguaje grafico utilizado para visualizar, especificar, construir y documentar sistemas de software y objetos distribuidos.

Las fuentes tecnicas investigadas incluyen:
- metamodelos UML
- sintaxis abstracta
- diagramas estructurales
- diagramas de comportamiento
- modelos XMI orientados a lectura automatizada

Dentro del proyecto se aplicaron:
- diagramas de clases
- diagramas de secuencia
- diagramas de actividades
- diagramas de estados

---

## Diagramacion UML y Lucidchart

Lucidchart incorpora herramientas de automatizacion mediante marcado UML (markup), permitiendo generar diagramas automaticamente a partir de texto.

Proceso tecnico:
1. habilitar biblioteca UML
2. seleccionar "Use Markup"
3. ingresar texto descriptivo
4. generar diagrama automaticamente

Esta funcionalidad facilita la documentacion tecnica y acelera la construccion de diagramas complejos.

## 2. Tecnologias Emergentes y Computacion

Las investigaciones tambien abarcaron conceptos tecnologicos modernos relacionados con infraestructura y computacion.

### Modelos de servicios en la nube

#### IaaS
Infrastructure as a Service proporciona infraestructura virtual y servidores mediante internet.

#### PaaS
Platform as a Service ofrece un entorno de desarrollo completo para crear y administrar aplicaciones.

#### SaaS
Software as a Service entrega aplicaciones listas para usar desde la nube mediante navegador web.

---

## Comparacion de modelos

- IaaS entrega infraestructura
- PaaS entrega herramientas de desarrollo
- SaaS entrega software terminado
